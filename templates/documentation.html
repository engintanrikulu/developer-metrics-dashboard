PR Throughput & Performance

PR Throughput

Description: Total number of pull requests created and merged within the selected time period.

Formula: Created PRs + Merged PRs

Optimization: Now fetches 20 PRs per repository (reduced from 30) for better performance

Example: 45 PRs created, 38 merged = 83 total throughput

Average MR Time

Description: Mean time from pull request creation to merge.

Formula: Sum(merge_time - created_time) / merged_PRs

Processing: Calculated using parallel data fetching

Example: 2.3 days average time from PR creation to merge

First Commit to Merge

Description: Time from the first commit in the PR to final merge.

Formula: merge_time - first_commit_time

Enhancement: Async commit data retrieval for faster calculation

Example: 4.1 days from first commit to merge

PR Size Analysis

Description: Distribution of PR sizes based on lines changed.

Processing: Parallel analysis of PR diffs and statistics

Small: <100 lines Medium: 100-500 lines Large: >500 lines
